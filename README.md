# DevForge Premium

Site premium da DevForge para apresentar projetos digitais focados em crescimento empresarial.

## Oferta

A DevForge cria projetos para empresas que precisam vender melhor, automatizar rotinas e organizar dados:

- landing pages de conversao
- sistemas sob medida
- banco de dados e dashboards
- automacao com IA
- seguranca, permissoes, logs e backups
- integracoes com WhatsApp, CRM, pagamentos, ERP, BI e redes sociais

## Rodar local

```bash
npm install
npm run serve
```

Abra `http://127.0.0.1:4173/`.

## Testes

```bash
npm test
```

## Deploy

Projeto pronto para Vercel com `vercel.json`.

```bash
vercel --prod --yes
```
