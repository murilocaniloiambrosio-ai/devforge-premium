# Prompts de Video IA DevForge Premium

Use estes prompts no Higgsfield depois de autenticar com:

```bash
higgsfield auth login
```

## Prompt 1: Computador descendo e sistema abrindo

Formato: vertical 9:16, 8 a 12 segundos.

Prompt:

Um computador premium desce lentamente em uma sala escura moderna, luzes ciano e laranja, camera dolly-in cinematografica. A tela liga mostrando um dashboard DevForge com leads, WhatsApp, banco de dados, IA, seguranca e graficos de crescimento. Particulas digitais saem de icones de TikTok, Instagram e WhatsApp e entram no sistema. Movimento elegante, alta energia, visual SaaS premium, sem parecer template, texto curto na tela: "Sua empresa aparece, responde, organiza e cresce". Final com logo DF e CTA: "Peca um diagnostico DevForge".

## Prompt 2: Nao e so site

Formato: vertical 9:16, 8 a 12 segundos.

Prompt:

Video publicitario futurista para uma empresa de tecnologia chamada DevForge Premium. Comeca com um celular recebendo mensagens perdidas no WhatsApp, planilhas baguncadas e leads sem resposta. Em seguida tudo se transforma em uma landing page clara, sistema interno, CRM, automacao com IA, banco de dados seguro e dashboard de decisao. Ritmo rapido, transicoes fluidas, luz neon ciano e laranja, camera com zoom suave, sensacao de produto caro e confiavel. Texto na tela: "Nao e so site. E fluxo de venda, dados e automacao".

## Prompt 3: IA com controle humano

Formato: vertical 9:16, 8 a 12 segundos.

Prompt:

Anuncio vertical premium mostrando uma IA empresarial organizada, sem exagero. Um painel DevForge recebe perguntas de clientes, classifica lead quente, resume conversas, cria follow-up e registra tudo em banco de dados seguro. Mostrar escudo de seguranca, logs, permissoes e aprovacao humana. Visual elegante, cinematic tech, interface realista, cores preto, ciano, verde e laranja, movimento de linhas de dados conectando WhatsApp, Instagram, TikTok e dashboard. Texto: "IA boa acelera a equipe sem tirar o controle".

## Comando base sugerido

```bash
higgsfield generate create seedance_2_0 --prompt "<cole o prompt>" --duration 8 --aspect_ratio 9:16 --wait
```

Se for usar Marketing Studio para anuncio com avatar:

```bash
higgsfield generate create marketing_studio_video --prompt "<cole o prompt>" --mode product_showcase --duration 15 --aspect_ratio 9:16 --resolution 720p --wait
```
